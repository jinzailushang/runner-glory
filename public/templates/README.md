# 马拉松模版图片

这个目录存放AI换脸的底图模版。

## 模版要求

### 图片规格
- 分辨率：1024x1024 或更高
- 格式：JPG 或 PNG
- 背景：透明或纯色背景，便于AI换脸

### 模版分类

#### 1. 冲线时刻 (finish)
- 运动员正在冲过终点线的姿态
- 包含奖牌、计时器等元素
- 男/女各2套

#### 2. 奖牌时刻 (medal)
- 运动员佩戴奖牌的姿态
- 突出奖牌和成就感
- 男/女各2套

#### 3. 赛道风景 (track)
- 赛道上的奔跑姿态
- 展现运动的动态美
- 中性设计，适合所有人

#### 4. 胜利手势 (victory)
- V字手势或其他胜利庆祝
- 展现喜悦和成就
- 中性设计，适合所有人

## Midjourney Prompt 示例

```
A handsome asian male runner crossing the finish line of a marathon, sunny day, professional sports photography, depth of field, wearing Nike running tank top, sweating, joyful expression, 4k, --ar 3:4
```

## 文件命名规范

- `finish-male-1.jpg` - 冲线男性1
- `finish-female-1.jpg` - 冲线女性1
- `medal-male-1.jpg` - 奖牌男性1
- `track-1.jpg` - 赛道1
- `victory-1.jpg` - 胜利1

对应的缩略图使用 `-thumb` 后缀。

## 注意事项

1. 确保图片中人物面部清晰可见，便于AI识别
2. 选择光线充足的场景，保证面部照明良好
3. 避免复杂背景，保持主体突出
4. 选择正面或3/4侧面的拍摄角度
